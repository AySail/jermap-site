Jer Maps

A timeline-based, embeddable, open-source world map

🌐 View the Project on GitHub

Key Features

🕒 Historical Timeline

Travel through time — see country names and borders from 753 BC to now.

📡 Satellite View

See realistic, earth-inspired satellite images. Supports month selection too.

🧬 Pangaea Mode

Explore a realistic satellite-style rendering of Pangaea created from scratch.

🧪 Ant/Bee/Wasp Colonies

See insect colonies around the globe. All data is open, and community-supported.

🧭 Community Naming

Users can suggest names and help shape how regions, borders, and features appear.

📦 Embed + API

Use Jer Maps for free in your own apps and websites. Just like Google Maps, but open-source!

Why Jer Maps?

Jer Maps isn’t just another map. It’s a creative, community-led experiment in visualizing history and geography. Built by an indie dev (that’s me 👋), it's completely open-source and ready to be part of your games, tools, and learning experiences. And yes — it’s free forever.
